A fully client-side, privacy-first ATS-Friendly Resume Builder designed for students and freshers. Create professional resumes with live preview, ATS scoring, multiple templates, auto-save functionality, and PDF export — all without creating an account or sharing personal data.

## Features

* 3 professional resume templates

  * Two-Column Professional
  * Single-Column ATS-Friendly
  * Modern Student Template
* Live A4 resume preview
* ATS Resume Score (0–100)
* Smart improvement suggestions
* Resume completion checklist with progress tracking
* Dynamic sections:

  * Education
  * Skills
  * Projects
  * Experience
  * Certifications
  * Achievements
* PDF export
* Local Storage auto-save
* Dark / Light mode
* Keyboard shortcuts
* Fully responsive design

## Why I Built This

While applying for internships, I frequently had to update and customize my resume. Existing solutions were often limited, complicated, or required account creation. I wanted a simpler and faster way to build ATS-friendly resumes, so I created ResumeAI.

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)
* Local Storage API
* Browser Print API

## Run Locally

Simply open `index.html` in any modern browser.

Or run a local server:

```bash
npx serve .
```

## Deploy to Vercel

1. Push project to GitHub
2. Import repository into Vercel
3. Select "Other" framework preset
4. Click Deploy
